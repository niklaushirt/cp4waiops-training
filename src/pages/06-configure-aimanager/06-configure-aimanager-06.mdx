In this section you will create the connection to Ansible Tower and create your first Runbook.


## üöÄ TASK: Create the Ansible Tower Connection

<a href="https://www.ibm.com/docs/en/cloud-paks/cloud-pak-watson-aiops/3.5.0?topic=connections-ansible" target="_blank" >üåè Documentation for the Task: Ansible automation provider</a>




Create the connection to Ansible Tower using the following parameters:

```yaml
URL for REST API:      from MY_CONFIGURATION.txt
Authentication type:   User ID/Password
User:                  admin
Password:              from MY_CONFIGURATION.txt
```





<Accordion>
<AccordionItem title="üí° Need Help?">





1. In the `CP4WAIOPS` "Hamburger" Menu select `Define`/`Data and tool connections`
1. Click `Add connection`
1. Under `Ansible Automation Controller`, click on `Add connection`
1. Click `Connect`

1. üîé Get the data from your configuration file `MY_CONFIGURATION.txt` - section  `2.3 Configure Runbooks - Ansible Automation Controller`

    ```bash
    ./tools/11_fzth/get_configuration_info.sh > MY_CONFIGURATION.txt
    ```

1. Fill out the fields:

	![image](./../images/29_runbook.png)

	```yaml
	URL for REST API:      from MY_CONFIGURATION.txt
	Authentication type:   User ID/Password
	User:                  admin
	Password:              from MY_CONFIGURATION.txt

1. Click `Done`

1. Make sure that the Connection Status turn green after a few seconds

## Check integration

## üöÄ TASK: Create a runbook

<a href="https://www.ibm.com/docs/en/cloud-paks/cloud-pak-watson-aiops/3.5.0?topic=library-create-runbook" target="_blank" >üåè Documentation for the Task: Creating a runbook</a>

Create a Runbook based on the `CP4WAIOPS Mitigate Robotshop Ratings Outage` Ansible Playbook





<Accordion>
<AccordionItem title="üí° Need Help?">




1. Select tab `Runbooks`
1. Click `Create Runbook`
1. Name it `RobotShop Mitigate MySQL Problem`
1. Click `Add automated step`
1. Select `CP4WAIOPS Mitigate Robotshop Ratings Outage`
1. Click `Select this action`
1. Click `Mapping` / `Select`
1. Select `New runbook parameter`

	![image](./../images/31_runbook.png)

1. Select tab 

1. üîé Get the data from your configuration file `MY_CONFIGURATION.txt` - section  `2.4 Configure Runbooks - Runbook Parameters` 

    ```bash
    ./tools/11_fzth/get_configuration_info.sh > MY_CONFIGURATION.txt
    ```

	![image](./../images/32_runbook.png)

1. Replace `PROVIDE: my_k8s_apiurl and my_k8s_apikey` in field `Default value (optional)` with the value from the script

1. Click `Save`
1. Click `Save` again
1. Click `Actions` and `Publish`

</AccordionItem>
</Accordion>
