# Demonstration summary
Today, I have shown you how Cloud Pak for Watson AIOps can assist the SRE/Operations team to identify, verify, 
and ultimately correct an issue with a modern, distributed application running in a cloud-native environment. 
The presented solution provides automatic application topology discovery, anomaly detection both with metrics and logs, 
and sophisticated methods of correlation of events coming from different sources.
