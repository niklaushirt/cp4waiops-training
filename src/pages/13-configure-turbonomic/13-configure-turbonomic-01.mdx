## üöÄ STEP: Connect to Turbonomic

<a href="https://www.ibm.com/docs/en/tarm/8.7.5?topic=clusters-installing-using-red-hat-openshift-operatorhub" target="_blank" >üåè Documentation for the Task:  Get the Route to open Turbonomic (Start at Step 9.)</a>


1. Click on `Networking/Routes`

    ![turbo](./../images/turbonomic/20_turbo.png)

1. Make sure you have selected the `turbonomic` Project

    ![turbo](./../images/turbonomic/21_turbo.png)

1. Click on the link for the `nginx` Route

    ![turbo](./../images/turbonomic/22_turbo.png)

1. The Turbonomic Login will open


<InlineNotification kind="warning"> 

**Unsafe Connection**

Depending on your Browser you might get a warning, saying that your connection is not secure.

Tell the browser to ignore and continue.

If you are using Chrome there might be no button to continue.

Just type "thisisunsafe" (without the quotes). there is no visual feedback while you type but if you typed it correctly, Chrome will continue to Turbonomic.

</InlineNotification>





## üöÄ STEP: Define Administrator Password

<a href="https://www.ibm.com/docs/en/tarm/8.7.5?topic=clusters-installing-using-red-hat-openshift-operatorhub" target="_blank" >üåè Documentation for the Task:  Define Administrator Password (Start at Step 10.)</a>


In this section you will define the Administrator Password



1. Enter the Password that you want to give the Turbonomic Administrator (twice)

    ![turbo](./../images/turbonomic/23_turbo.png)

1. Click `Create Account`

1. Click `Save And Continue`

    ![turbo](./../images/turbonomic/24_turbo.png)

1. You will get a warning that there is no Turbonomic license loaded

    ![turbo](./../images/turbonomic/25_turbo.png)

1. Continue to next Task to load a license

## Activate Turbonomic

1. Go to the following page:

    <a href="https://ibm.biz/training-activation" target="_blank" >üöÄ Training Activation Portal</a>

2. Login with the Access Information **provided by the Course Trainer**:
    - Course ID
    - EMail Address
    - Course Password

    ![instana](./../images/instana/instana_install_02.png)

2. Click on `Activate Turbonomic`

    ![instana](./../images/turbonomic/01_turbo_install.png)

2. Paste in the needed information
    - Turbonomic URL: copy from your browser
    - Turbonomic Admin User: usually administrator
    - Turbonomic Admin Password: The administrator password you have set in the previous step

3. Click `Activate`

    ![instana](./../images/turbonomic/02_turbo_install.png)

2. If all is good, you should get a success message
